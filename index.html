Discussion of Inline/block elements

  • Inline - img, a, br, em, string, span
  • Block elements - p, h1, ul, li

Tags discussed during session

  • header (within body) - Defines a header for a document or section; General rule of thumb: Logo, Nav, search bar
  • main - Specifies the main content of a document
  • aside - Defines content aside from the page content
  • footer - Defines a footer for a document or section
  • span - Can be used to highlight content; however look at OTHER relevant tags i.e. strong or em

Pseudo-classes

Can be used with the anchor (a) tag for hyper-links

  • a:link {color:#FF0000;} /* unvisited link */
  • a:visited {color:#00FF00;} /* visited link */
  • a:hover, a:focus {color:#FF00FF;} /* mouse over or select with keyboard*/
  • a:active {color:#0000FF;} /* selected link */

Overview of the box model

  • Covering the contend, padding, margin.
  • Insight into responsive design using the calc() function with the width selector.

2.3 - HTML & CSS (continue) and 2.4 - Positioning and layout

What have I learnt?

Box model (from 2.3)

  • Content - Actual content
  • Padding - Area between content and board
  • Border -
  • Margin - Space between one element and another

padding:

  • padding: 15px; // 15 pixels on all sides
  • padding-top: 10px; // padding top 10px
  • padding: 10px; 20px; 30px; 40px; // top right bottom left
  • Note: The order is clockwise: top right bottom left

border:

  • border: 1x solic #ff0000; // a solid red boarder
  • boarder-top: 4px dotted #000000; think dotted border
  • border-width: 10px; border 10px wide
  • border-style: dashed; // border dashed
  • border-color: #666666; // colour dark grey

Margins are the same as padding
